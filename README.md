# shangchen商城系统
***
## 框架搭建
* 使用Springboot+Mybatis搭建基本骨架
* 使用Swagger-UI实现在线API文档
* 整合了Redis实现缓存功能
* 整合了SpringSecurity和JWT实现认证
* 整合了SpringTask实现定时任务
* 整合了MongoDB实现文档操作
